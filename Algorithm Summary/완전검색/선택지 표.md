# 순열

## 중복 순열

- 그냥 for문 r번 `n^r`

## 순열

- n개중 r개를 뽑아 순서있게 배치
- `nPr`
- 반복문: for문 중복체크, 중복시 continue
- 재귀: 중복체크 isSelected[] = true ?, 중복시 continue

# 조합

## 중복 조합

- `nHr` = `n+r-1Cr` 칸막이 배치를 생각하기!
- 반복문: 선택한 수 부터 탐색가능! j = i
- 재귀: 선택한 수부터 탐색 가능! start = i

## 조합

- 그냥 조합
- `nCr`
- 반복문: 선택 다음부터 탐색! j = i+1
- 재귀: 선택한 수 다음부터 탐색! start = i+1
