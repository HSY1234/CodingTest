# 시간 복잡도

약 1억번 연산 ≒ 1초

|      N의 크기      | 허용 시간복잡도 |
| :----------------: | :-------------: |
|       N ≤ 11       |      O(N!)      |
|       N ≤ 25       |     O(2^N)      |
|      N ≤ 100       |     O(N^4)      |
|      N ≤ 500       |     O(N^3)      |
|      N ≤ 3000      |   O(N^2logN)    |
|      N ≤ 5000      |     O(N^2)      |
| N ≤ 1000000(백만)  |    O(NlogN)     |
| N ≤ 10000000(천만) |      O(N)       |
|      그 이상       |  O(logN), O(1)  |
