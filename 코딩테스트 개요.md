# 코딩테스트 개요

- 코딩테스트는 주어진 문제를 정해진 시간 제한과 메모리 제한 내에 해결할수 있는 능력을 측정하는 테스트

## 코딩테스트 능력

- 배경지식
  - 자료구조
  - 알고리즘
- 문제해결 능력
  - 배경지식을 문제에 맞게 변형하고 적용하는 능력
- 구현력
  - 본인이 생각한 풀이를 잘 옮겨내는 능력
