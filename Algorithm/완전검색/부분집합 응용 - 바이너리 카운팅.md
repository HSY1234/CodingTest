# 부분집합 응용

- 집합에 포함된 원소들을 선택하는것
- 바이너리 카운팅을 통한 사전적 순서로 생성하는 방법은 모든 부분집합을 찾는 가장 자연스러운 방법

```java
int[] arr = {3,6,7,1,5,4};
int n = arr.length;

for(int i=0, i<(1<<n)); i++){ // 부분 집합 개수만큼 2^n
  for(int j=0; j<n;j++){
    if(i&(1>>j)!=0){
      System.out.print(arr[j]+" ");
    }
  }
  System.out.println();
}

```
